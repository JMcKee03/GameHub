# JavaScript Game Hub

A browser-based mini game hub featuring Tic-Tac-Toe, Rock Paper Scissors, and Connect 4. Built with vanilla JavaScript, HTML, and CSS, this project focuses on clean game logic, responsive UI, and smooth user interaction without external libraries.

Each game is fully playable in the browser and designed with reusable UI patterns and clear state handling to ensure predictable and engaging gameplay.

## Features
- Three playable games in a single hub
  - Tic-Tac-Toe
  - Rock Paper Scissors
  - Connect 4
- Clear win, loss, and tie detection
- Real-time UI updates using DOM manipulation
- Reset and replay functionality
- Responsive layout for desktop and mobile

## Technologies Used
- JavaScript (ES6+)
- HTML5
- CSS3
- DOM manipulation and event handling
